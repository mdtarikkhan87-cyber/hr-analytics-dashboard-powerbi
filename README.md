 📊 HR Analytics Dashboard (Power BI)

📌 Overview

Built an interactive HR dashboard to analyze employee distribution, salary patterns, and workforce performance using Power BI.



 🎯 Business Problem

Organizations need visibility into employee distribution, compensation, and performance to make better HR decisions.



🛠️ Tools Used

* Power BI
* Microsoft Excel


📸 Dashboard Preview

![Dashboard](images/dashboard.png)



📊 Key Metrics

* Total Salary: ₹5.16M
* Total Performance Score: 160
* Employee Trend (Year-wise)


 📈 What I Did

* Designed interactive dashboard with filters (Joining Date, Exit Date)
* Analyzed salary distribution by department, designation, and city
* Visualized employee trends and workforce structure
* Built KPI cards for quick decision-making



🔍 Key Insights

* High salary concentration in specific departments
* Certain cities contribute more to workforce cost
* Hiring trend shows steady growth
* Performance distribution highlights overall workforce efficiency
